Install
=======

create conda
------------

First , create a virtual environment (recommend anaconda)

.. code-block:: python

    conda create -n deepke python=3.8
    conda activate deepke

install by pypi
---------------

If use deepke directly

.. code-block:: python

    pip install deepke


install by setup.py
-------------------

If modify source codes before usage

.. code-block:: python

    python setup.py install
